Project Overview

Bitcoin Price Prediction Using Machine Learning is a comprehensive Python-based web application that leverages advanced machine learning algorithms to analyze historical Bitcoin price data and generate accurate future price predictions. The platform combines technical analysis, statistical modeling, and modern web technologies to provide traders and investors with data-driven insights.

The application features a complete user management system, CSV data upload and analysis capabilities, real-time visualization dashboards, and multiple machine learning models including Random Forest, Linear Regression, and Support Vector Regression. All wrapped in a modern, responsive web interface with dark/light theme support.
